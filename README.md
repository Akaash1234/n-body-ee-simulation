# N-Body Simulation & Analysis Tool

A comprehensive N-Body gravity simulation dashboard built with Python and Streamlit.

## Features
- **Multiple Integrators**: Euler, Verlet, Leapfrog, RK4
- **Lagrange Point Analysis**: Calculate and visualize L1-L5, stability analysis, and CR3BP trajectories
- **NASA Data**: Load real solar system data from JPL Horizons
- **Interactive Visualization**: 2D/3D plots and animations
- **Physics Analysis**: Real-time energy and angular momentum tracking

## Usage
```bash
pip install -r requirements.txt
streamlit run app.py
```
